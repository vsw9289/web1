Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a aeb server or from local storage and render them into miltimedia web pages. HTML deceribes the structure of a web page semantically and originally included cuse for the appearance of the document.

HTML elements are the blilding blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
